# G-2140: Never initialize variables with NULL.

!!! tip "Minor"
    Maintainability

## Reason

Variables are initialized to NULL by default.

## Example (bad)

``` sql
DECLARE
   l_note big_string_type := NULL;
BEGIN
   sys.dbms_output.put_line(l_note);
END;
/
```

## Example (good)

``` sql
DECLARE
   l_note big_string_type;
BEGIN
   sys.dbms_output.put_line(l_note);
END;
/
```
