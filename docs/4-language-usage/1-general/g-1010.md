# G-1010: Try to label your sub blocks.

!!! tip "Minor"
    Maintainability

## Reason

It's a good alternative for comments to indicate the start and end of a named processing.

## Example (bad)

``` sql
BEGIN
   BEGIN 
      NULL;
   END;

   BEGIN
      NULL;
   END;
END;
/
```

## Example (good)

``` sql
BEGIN
   <<prepare_data>>
   BEGIN 
      NULL;
   END prepare_data;

   <<process_data>>
   BEGIN
      NULL;
   END process_data;
END good;
/
```
